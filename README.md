# 💸 BudgetTracker

A sleek, dark-themed, and minimalist personal finance manager crafted for speed and absolute privacy. No servers, no logins—just pure local control over your data.

Developed and maintained by **NoviumNodes**.

---

## ⚡ Features

* **🧠 Persistent Memory:** Uses the browser's `LocalStorage` to save your financial data automatically. Your data never leaves your device.
* **📥 JSON Portability:** Full support to **Export** your complete history as a `.json` file, or **Import** a backup to restore your data instantly.
* **📊 Visual Tracking:** Clean structure to monitor your daily income, expenses, and net balance at a glance.
* **🎯 Minimalist Dark UI:** Easy on the eyes, built with semantic HTML, modern CSS layout grids, and optimized for performance.

---

## 🛠️ Tech Stack

* **Structure:** HTML5 (Semantic elements)
* **Styling:** CSS3 (Custom properties, Flexbox, and CSS Grid)
* **Logic:** Vanilla JavaScript (ES6+, LocalStorage API, Blob API for JSON handling)

---

## 🚀 How to Use Globally

Since this project runs entirely on **Vanilla JavaScript**, you can run it instantly without any local setup:

1. Visit our live page via GitHub Pages (Link here).
2. Start adding your daily transactions.
3. Use the **Export** button to download your data as a backup `.json` file before clearing your browser cache.
4. Use the **Import** button to upload your `.json` file anytime to resume tracking.
